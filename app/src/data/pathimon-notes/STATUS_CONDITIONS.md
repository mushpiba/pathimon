# 상태이상 목록

이 상태이상들은 패시몬이 보스/사람 적에게 부여하는 전투 상태이상이다. 같은 상태이상은 누적 가능하며 UI에는 `발열(3)`처럼 표시한다.

| 상태이상 | 효과 |
|---|---|
| 발열 | 매 턴 최대 체력 2% 피해 |
| 탈수 | 받는 직접 피해 10% 증가 |
| 피로 | 공격력 10% 감소 |
| 구토 | 체력 회복량 10% 감소 |
| 배설 이상 | 턴 종료 시 20% 확률로 `탈수` 1스택 추가 |
| 기침 | 보스가 공격할 때마다 현재 체력 2% 피해 |
| 혈압 이상 | 상태이상으로 받는 피해 10% 증가 |
| 호흡 곤란 | 즉사 확률 1% 증가 |
| 부종 | 모든 상태이상의 확률 1%p 증가 |
| 신경 이상 | 혼란 확률 10%p 증가 |
| 마비 | 마비 확률 5%p 증가 |
| 출혈 | 매 턴 현재 체력 2% 피해 |
| 면역 이상 | 2스택마다 방어 특성 1개 무력화 |
| 괴사 | 최대 체력 상한 5% 감소 |
| 시력 이상 | 명중률 25% 감소 |
| 청력 이상 | 명중률 25% 감소 |
| 통증 | 매 턴 최대 체력 2% 피해 |
| 가려움 | 마비 확률 2%p 증가 |
| 황달 | 받는 직접 피해 5% 증가 |
