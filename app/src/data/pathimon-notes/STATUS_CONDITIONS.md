# 상태이상 목록

이 상태이상들은 패시몬이 보스/사람 적에게 부여하는 전투 상태이상이다. 같은 상태이상은 누적 가능하며 UI에는 `발열(3)`처럼 표시한다.
정본은 `claudecode/VOCAB.md` §3이고, 구현은 `app/src/data/statusConditions.ts`다.

| 상태이상 | 효과 | 계열 |
|---|---|---|
| 발열 | 매 턴 최대 체력 2% 피해 | 도트 |
| 출혈 | 매 턴 현재 체력 2% 피해 | 도트 |
| 빈혈 | 매 턴 최대 체력 1% 피해 | 도트 |
| 배설 이상 | 매 턴 최대 체력 1% 피해, 턴 종료 시 20% 확률로 `탈수` 1스택 추가 | 도트 |
| 기침 | 공격할 때마다 현재 체력 2% 피해 | 도트 |
| 괴사 | 최대 체력 상한 2.5% 감소 | 최대 HP |
| 피로 | 공격력 5% 감소 | 능력치 |
| 통증 | 방어력 5% 감소 | 능력치 |
| 혈압 이상 | 방어력 7.5% 감소, 상태이상으로 받는 피해 5% 증가 | 능력치 |
| 시력 이상 | 명중률 12.5% 감소 | 명중 |
| 청력 이상 | 명중률 7.5% 감소 | 명중 |
| 마비 | 행동 불가 확률 5% 증가 | 행동 |
| 구토 | 행동 실패 확률 4% 증가 | 행동 |
| 가려움 | 행동 실패 확률 2.5% 증가 | 행동 |
| 신경 이상 | 혼란 확률 5%p 증가 | 행동 |
| 탈수 | 체력 회복량 12.5% 감소 | 회복 |
| 황달 | 체력 회복량 7.5% 감소 | 회복 |
| 호흡 곤란 | 받는 직접 피해 5% 증가, 즉사 확률 0.5% 증가 | 메타 |
| 부종 | 모든 상태이상의 확률 1%p 증가 | 메타 |
| 면역 이상 | 4스택마다 방어 특성 1개 무력화 | 메타 |
