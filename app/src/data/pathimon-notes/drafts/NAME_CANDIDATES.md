# 1차 패시몬 이름 후보

이 파일은 1차 후보 59개만 대상으로 한다. 이전 129개 전체 후보표는 취소하고 이 표로 대체한다.

기준:

- 1~54번은 `docs/infection-immune-pathimon-candidates.md`의 1차 핵심 후보 기준이다.
- 55~59번은 draft에 후보 번호 56~60으로 추가되어 있던 항목 중 1차 확장 후보로 정리한 5개다.

- 사용자가 이름을 확정할 때는 한 줄에 하나씩 `번호-후보문자` 형식으로 적으면 된다.

예시:

```text
1-a
2-g
55-f
```
