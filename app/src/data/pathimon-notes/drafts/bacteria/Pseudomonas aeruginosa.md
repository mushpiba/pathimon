# 패시몬 노트

이름: 녹슬농
학명: 녹농균(Pseudomonas aeruginosa)
타입: 세균
태그:
- structure: 그람음성
- location: 상처, 폐
- evasion: 생물막, 협막
- pathway: 의료관련

학습포인트:
- L1 [감별점] 녹농균은 어디에나 있는 그람음성 막대균이다. 운동성이 있고 **절대호기성**이다. 오염된 물과 습도 높은 곳에 살며 병원감염 3대 원인균 중 하나다.
- L2 [감별점] **배양 동정 축은 셋이다 — 색소, 산화효소 양성, 42℃ 성장.** 청색 색소 피오시아닌과 형광 색소 피오버딘이 섞여 밝은 녹색을 낸다. 포도 같은 단 냄새가 난다.
- L3 [감별점] 장내세균과와 달리 **유당을 발효하지 않는다**(MacConkey 음성). 혈액한천에서는 β-용혈을 보인다.
- L4 [기전] 부착 인자는 편모·섬모·LPS다. 알긴산염 협막은 식세포 작용을 막고, 고체 표면 부착과 오랜 정착·생물막 형성을 돕는다.
- L5 [기전] **외독소 A는 EF-2를 ADP-리보실화해 불활성화한다.** 세포내섭취로 들어가 산성 조건에서 A 소단위가 세포질로 나오고, 단백합성이 멈춰 세포가 죽는다. 화상 부위 피부괴사, 각막 손상, 폐 조직 손상을 만든다. **디프테리아 독소와 같은 방식이다.**
- L6 [기전] 피오시아닌은 활성산소종을 만들어 조직을 손상하고, IL-8 분비를 늘려 호중구를 불러 모은다.
- L7 [기전] 엘라스타제·인지질분해효소·단백질분해효소가 조직을 부수고 보체를 분해한다. III형 분비계는 외독소 S와 T를 숙주 세포 안으로 직접 꽂아 넣는다.
- L8 [기전] **내성 기전이 여러 겹이다.** 포린 단백 변이로 항생제가 들어오지 못하고, β-lactamase가 β-lactam을 변형하며, 여러 다제 배출펌프가 β-lactam·플루오로퀴놀론·마크로라이드를 퍼낸다.
- L9 [감별점] 기회감염균이라 걸리는 자리가 정해져 있다 — 낭포성 섬유증 환자와 면역저하자의 기관기관지염, 화상 부위 감염과 모낭염, 유치도뇨관 요로감염, 수영자의 외이염, 콘택트렌즈 각막궤양, 균혈증과 심내막염, 그리고 괴저성 농창이다.
- L10 [감별점] 낭포성 섬유증은 염소이온 통로 단백의 양과 기능이 떨어져 두껍고 끈적한 점액이 만들어지는 상염색체 열성 질환이다. 그 점액에 녹농균이 정착한다.
- L11 [감별점] 괴저성 농창은 검은 가피 아래 농양이 생기고, 가피가 떨어지면서 경계가 융기된 공동성 궤양이 남는다.
- L12 [치료] 여러 계열에 본래부터 내성이라 **병용요법이 필요하다.** 증례에서는 감수성 결과에 따라 ceftazidime과 tobramycin으로 바꿨다.
- L13 [치료] **생물막을 만드는 기구연관 감염에서는 항생제만으로 안 된다.** 인공호흡기 같은 오염된 기구의 교체·제거와 소독을 반드시 병행한다.

- 판단 기록:
  - v1 노트는 `이름:`이 공란이고 학습포인트·대처법이 통째로 없었다. v2 규격으로 새로 썼다
  - 이름 `녹슬농`은 사용자가 후보 5개 중 선택했다
  - v1 `location: 습윤환경/호흡기/상처` → `상처, 폐`. 습윤환경은 서식지이지 침범 부위가 아니라 학습포인트와 대처법으로 옮겼다. `pathway: 병원내/기회감염` → `의료관련`
  - v1 `방어특성: 바이오필름` → `evasion: 생물막, 협막`. VOCAB §2-5 정규화표대로 표기를 고치고, 알긴산염 협막이 슬라이드 14에서 따로 다뤄지므로 협막을 더했다. 생물막이 정체성이라 첫 값으로 뒀다
  - **방어 92는 CRE(95) 다음으로 높다.** 획득 내성이 아니라 포린·배출펌프에 의한 **본래 내성**이라는 점이 강의의 강조점이고, 단일 약이 아니라 병용요법과 기구 제거를 요구한다는 점에서 밴드 5가 맞다
  - **전용기 `생물막 장벽`의 v1 효과 `무적(3)`을 `반감(2)`로 낮췄다.** 3턴 무적은 밸런스상 과하고, 생물막은 약을 완전히 막는 게 아니라 닿기 어렵게 하는 구조다. 무효/금기의 "항생제 단독요법"과도 맞물린다
  - `조직 분해`를 확률 분기로 둔 것은 슬라이드 15·19가 피오시아닌(ROS·IL-8)과 분해효소·III형 분비계를 다른 병독인자로 나눠 표시하기 때문이다
  - v1의 `면역 이상(2)`은 뺐다. 면역 이상은 방어 특성을 무력화하는 상태인데, 녹농균의 생물막은 오히려 자기 방어를 세우는 쪽이라 방향이 반대였다
  - 강의가 다뤘으나 학습포인트에 넣지 않은 것: 형광 색소 유무로 나눈 Pseudomonas 종 목록(슬라이드 11), 교수의 싱크대 배양 일화. 종 목록은 P. aeruginosa 감별에 직접 쓰이지 않는다
  - 기술↔학습포인트 대응: 습윤환경 정착 → L1·L4·L9 / 외독소 A → L5 / 조직 분해 → L6·L7 / 생물막 장벽 → L4·L8·L13
