# 보스 공격-패시몬 방어 상성 설계

## 목표

보스전은 `무효 / 보통 / 굉장함` 3단계만 사용한다. 보스는 현재 대면 중인 패시몬에게 효과적인 공격을 고르지만, 플레이어가 교체로 그 공격을 무효화하면 해당 보스 기술이 봉인된다. 보스 기술 4개가 모두 봉인되면 보스는 1턴을 정비에 쓰고, 새 무작위 기술 4개를 장착한다.

## 판정 단위

보스 공격 판정은 기존 플레이어 공격용 `EFFECTIVENESS`와 분리한다. 보스 공격은 별도 테이블에서 `none`, `normal`, `super` 중 하나로 판정한다.

패시몬 방어 프로필은 아래 자료를 합친다.

- 명시 방어특성: 패시몬 노트의 `방어특성`
- 태그: `structure`, `location`, `pathway` 등에서 파싱된 태그
- 타입 파생 특성: 세균, 바이러스, 진균, 원생동물, 기생충
- 자동 파생 특성: 세포벽, 바이러스성, 진균막, 대형 병원체, 독소축 등

`면역 이상` 2스택마다 명시 방어특성 1개를 앞에서부터 무력화한다. 태그와 타입 파생 특성은 질병 정체성에 가까우므로 면역 이상으로 제거하지 않는다.

## 보스 턴 규칙

1. 보스는 기술 4개를 가진다.
2. 보스는 플레이어가 행동하기 전에 현재 패시몬을 보고 공격을 예고한다.
3. 예고된 기술은 플레이어가 교체해도 바뀌지 않는다.
4. 공격이 실제 대상에게 `무효`면 피해와 효과를 적용하지 않고 그 기술을 봉인한다.
5. 공격이 `굉장함`이면 피해 배율 2배, `보통`이면 1배를 적용한다.
6. 봉인되지 않은 기술 중 현재 패시몬에게 `굉장함`인 기술이 있으면 그중 무작위로 고른다.
7. `굉장함`인 기술이 없으면 `보통` 기술 중 무작위로 고른다.
8. 모든 기술이 봉인되면 다음 보스 행동은 정비가 되며, 피해 없이 새 무작위 기술 4개를 얻고 봉인을 해제한다.

## 노트 작성 규칙

패시몬 노트에는 가능한 한 `태그`와 `방어특성`을 비워두지 않는다.

- 실제로 특이 방어기전이 없으면 `방어특성: 없음`으로 명시한다.
- 세포벽이 없으면 `방어특성: 무세포벽`을 넣는다.
- 협막, 아포, 바이오필름, 항산성, 항원변이, 항원위장, 세포내생존, 잠복, 대형저항은 우선적으로 방어특성에 반영한다.
- `structure`는 그람양성/그람음성/항산성/피막바이러스/레트로바이러스/진균/선충/흡충/조충처럼 보스 공격 판정에 쓰일 값을 넣는다.
- `location`은 세포외/세포내/세포질/식포/적혈구/장관강/혈관 등으로 채운다.
- `pathway`는 감염경로 학습과 일부 장벽 판정에 쓰이므로 가능한 한 채운다.
