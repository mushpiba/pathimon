# PATHOGEN TOWER — 전투 시스템 정리 v3 (효과/증상 분리 · 상태이상 사전)

> v2에서 확정 반영:
> 1. **표적태그를 공격기에서 제거 → 병원체 태그(`location`)로 이동.** 공격기는 태그를 참조만.
> 2. **효과(전투 로직) vs 증상(표시·학습) 분리.** 이 게임의 핵심 재미 = 상태이상·증상을 쌓아 "감염 서사"를 만드는 것.
> 3. 상태이상은 병태생리 기반 사전으로 관리. 증상은 순수 텍스트.

---

## 0. 전투 구도 (확정)

- **패시몬(공격) ↔ 사람(방어)** / **사람(공격) ↔ 패시몬(방어)**. 패시몬끼리 전투 없음.
- 두 방향 독립 상성표:
  - **표1** 패시몬 공격타입 × 사람 방어특성
  - **표2** 사람 공격타입 × 패시몬 방어특성 (+ 패시몬 태그)
- 면역계 = **사람의 공격**. 사람 방어특성 = 면역 제외(1차 장벽 + 수동저항).

---

## 1. 공격기 데이터 구조 — 4정보

패시몬 공격기 하나는 **딱 4가지**만 담는다. (표적태그는 병원체 태그에서 참조, 공격기엔 없음)

| 정보 | 필드 | 역할 |
|---|---|---|
| ① 위력·명중 | `power` `acc` | 데미지 계산 |
| ② 타입(관통) | `type` | 사람 방어특성과 상성 판정 (물리파괴/독소/면역매개/특수) |
| ③ 효과 | `effects[]` | **전투 로직** — 실제 상태이상 부여(데미지/디버프) |
| ④ 증상 | `symptom` | **표시·학습용 텍스트** — 전투 영향 0 |

```js
{ id:'cholera_toxin', name:'콜레라독소',
  power:16, acc:85,
  type:'독소',
  effects:['탈수'],            // 전투 상태이상(로직)
  symptom:'쌀뜨물 설사'         // 표시·학습 텍스트
}
// location·structure 등은 병원체 태그에서 참조 → 공격기엔 중복 기재 안 함
```

### 효과 vs 증상 — 분리 원칙
- **효과(effects)**: 전투에 실제 영향. 상태이상 사전(§3)의 키를 참조.
- **증상(symptom)**: 화면·도감 표시용 텍스트. 전투 수치 0.
- **누적 표시**: 공격할 때마다 증상이 로그/사이드에 스택 → "발열 → 쌀뜨물 설사 → 탈수 쇼크"로 병이 진행되는 서사. 전투 종료 시 "감염 경과" 요약.
- 학습 효과: 증상 텍스트가 곧 암기 카드(콜레라=쌀뜨물 설사).

## 2. 준비기 (Preparation move)

- 공격기가 아니라 **감염 정착(부착·침입·정착)** 을 표현하는 랭크업 버프기.
- 효과 = 자기 스탯 상승. **교체 시 초기화**(포켓몬 랭크업식). 현재 전투 한정.
- 콜레라 예시: 「소장 상피 부착」 = 남은 전투 공격력 2배.

```js
{ id:'cholera_attach', name:'소장 상피 부착', kind:'prep',
  symptom:'소장 상피세포 표면에 부착',
  effects:[ {kind:'buff', stat:'attack', mult:2.0, scope:'battle', target:'self'} ] }
```

> **미확정**: 준비기가 턴을 소모하는지(빌드업 vs 즉딜 선택 생성) — §8.

## 3. 상태이상 사전 (효과 = 전투 로직)

병태생리 기반. **원인(병이 만드는 것) → 전투 효과** 매핑. `effects[]`가 이 키를 참조.

| 상태이상 | 전투 효과 | 실제 의미 | 성격 |
|---|---|---|---|
| **발열** | 매 턴 체력 %독뎀(경미) | 전신 염증·대사 항진 | 지속 |
| **설사** | 체력 %독뎀(약) + 탈수 유발 | 장관 수분 손실 | 지속 |
| **탈수** | 체력 %독뎀 + 방어력↓ | 체액 손실→순환 저하 | 급성 |
| **쇼크(혈압저하)** | 큰 폭 체력 독뎀(위중) | 패혈증·탈수 말기 | 급성 치명 |
| **근육경련** | 2턴 스턴 | 파상풍 강직·전해질 이상 | 행동불가 |
| **마비** | 스턴(행동불가) | 보툴리눔·신경독 | 행동불가 |
| **기침** | 상대 명중↓ or 전파게이지↑ | 호흡기 배출 | 유틸 |
| **빈혈** | 공격력↓ + 회복 차단 | 적혈구·혈액 손실 | 디버프 |
| **출혈** | 지속 독뎀 | 조직·혈관 파괴 | 지속 |
| **뇌증(의식저하)** | 혼란(50% 자기공격) | 뇌말라리아·수막염 | 제어불능 |
| **면역붕괴** | 사람 공격력 지속↓ | HIV의 CD4 고갈 | 만성 누적 |
| **섬유화** | 상대 방어 고정↑(누적) | 만성 육아종·조직경화 | 만성 누적 |

**설계 원칙**
- 급성 치명(탈수·쇼크·마비) = 강한 즉효 / 만성(섬유화·면역붕괴) = 약하지만 지속·누적.
- 이 대비가 병원체 성격을 만든다: **콜레라(빠른 탈수사) vs 결핵(느린 붕괴)**.
- 혈압은 단독 상태이상이 아니라 **쇼크**로 통합(탈수·패혈증 심화 단계).

```js
STATUS['탈수'] = { effect:'hp_percent_dot+def_down', tickPct:8, defDown:20, icon:'💧', desc:'체액 손실로 방어·순환 저하' }
STATUS['근육경련'] = { effect:'stun', turns:2, icon:'⚡', desc:'강직성 경련으로 행동불가' }
```

## 4. 증상 (표시·학습용 텍스트)

순수 문자열. 전투 영향 없음. 병원체별 특징 증상 = 암기 카드.

| 병원체 | 대표 증상(symptom 텍스트) |
|---|---|
| 콜레라 | 쌀뜨물 설사 (rice-water stool) |
| 말라리아 | 주기적 발열·오한 (48/72h) |
| 결핵 | 만성 기침·야간발한·객혈 |
| 인플루엔자 | 고열·근육통·기침 |
| HIV | 기회감염·면역결핍 |
| 칸디다 | 구강 아구창·백태 |
| 아스페르길루스 | 객혈·폐 침윤 |
| 이질아메바 | 혈성 점액변·간농양 |
| 회충 | Löffler 증후군(폐이행 기침) |
| 주혈흡충 | 혈뇨·간 섬유화 |

## 5. 병원체 태그 (표적 포함, 여기로 이동)

공격기에서 뺀 표적태그는 여기 `location`으로. 병원체당 한 번.

- `structure`(정체성): 그람양성/음성/항산성/무벽 · 피막바이러스/핵산 · 진균벽(효모/균사) · 선충/흡충/조충/원충
- `location`(위치=옛 표적태그): 세포외 / 세포내(세포질=CTL) / 세포내(소포=Th1) / 절대세포내
- `pathway`(경로): 호흡기/소화기/혈액매개/피부/성접촉/접촉
- `size`(크기): 미시 / 대형(연충→호산구)
- `vector`·`oxygen`·`reservoir`·`stage`: 확장 축

판정 시 사람 공격이 이 태그를 참조: `항체 × location:세포내 = 무효`, `세포벽약물 × structure:무벽 = 무효`.

## 6. 대표 병원체 10종 통합 시트

각 항목: 태그 / 준비기 / 공격기(위력·명중·타입·효과·증상) / 방어특성.

### 1) 인플루엔자 (바이러스)
- 태그: `structure:피막바이러스(ssRNA−)` `location:세포내(세포질)` `pathway:호흡기` `size:미시`
- 준비기 「혈구응집소 부착」: 공격력↑ / 증상 "상피 부착·침입"
- 공격기 「뉴라미니다제 확산」 위력13 명중90 `type:물리파괴` 효과[발열,기침] 증상 "고열·근육통"
- 방어특성 「항원변이」: 사람 *항체·ADCC* 무효

### 2) HIV (바이러스·레트로)
- 태그: `structure:피막바이러스(retro)` `location:세포내(세포질)` `pathway:혈액/성접촉` `size:미시`
- 준비기 「gp120 결합」: 공·방↑ / 증상 "프로바이러스 삽입"
- 공격기 「CD4 사령부 강탈」 위력12 명중90 `type:특수` 효과[면역붕괴] 증상 "기회감염·면역결핍"
- 방어특성 「항원변이+잠복」: 사람 *항체·CTL* 반감/무효

### 3) 콜레라 (세균·그람음성)
- 태그: `structure:그람음성` `location:세포외` `pathway:소화기` `size:미시`
- 준비기 「소장 상피 부착」: **공격력 2배** / 증상 "소장 상피 부착"
- 공격기 「콜레라독소」 위력16 명중85 `type:독소` 효과[탈수,설사] 증상 "쌀뜨물 설사"
- 방어특성 「무방비」: 회피 없음(항체·포식에 약함)

### 4) 결핵 (세균·항산성)
- 태그: `structure:항산성` `location:세포내(소포)` `pathway:호흡기` `oxygen:호기` `size:미시`
- 준비기 「대식세포 잠입」: 방어·지속력↑ / 증상 "육아종 형성"
- 공격기 「만성 소모」 위력9 명중95 `type:면역매개` 효과[섬유화] 증상 "만성 기침·야간발한·객혈"
- 방어특성 「식포융합차단+활성산소중화」: 사람 *포식·산화* 무효

### 5) 칸디다 (진균·이형성)
- 태그: `structure:진균벽(이형성)` `location:세포외` `pathway:점막` `reservoir:상재균` `size:미시`
- 준비기 「균사 전환」: 공·방↑ / 증상 "효모→균사 전환"
- 공격기 「균사 관통」 위력11 명중92 `type:물리파괴` 효과[출혈] 증상 "구강 아구창·백태"
- 방어특성 「바이오필름」: 사람 *포식·약물* 반감

### 6) 아스페르길루스 (진균)
- 태그: `structure:진균벽(균사)` `location:세포외` `pathway:호흡기` `reservoir:환경` `size:미시`
- 준비기 「포자 발아」: 공격력↑ / 증상 "포자 발아"
- 공격기 「혈관 침습」 위력13 명중88 `type:물리파괴` 효과[출혈,쇼크] 증상 "객혈·폐 침윤"
- 방어특성 「활성산소중화」: 사람 *산화살균* 반감

### 7) 말라리아 (원충)
- 태그: `structure:원충` `location:세포내(적혈구)` `pathway:혈액매개` `vector:모기` `size:미시`
- 준비기 「적혈구 침입」: 공격력↑ + 주기버프 / 증상 "간·적혈구 증식"
- 공격기 「적혈구 동기파열」 위력15 명중85 `type:물리파괴` 효과[발열,빈혈] 증상 "주기적 발열·오한"
- 방어특성 「항원변이」: 사람 *항체* 반감 + 세포내 은신

### 8) 이질아메바 (원충)
- 태그: `structure:원충` `location:세포외(조직침습)` `pathway:소화기` `size:미시`
- 준비기 「탈낭·부착」: 공격력↑ / 증상 "대장 상피 부착"
- 공격기 「접촉 용해」 위력14 명중88 `type:물리파괴` 효과[출혈,설사] 증상 "혈성 점액변·간농양"
- 방어특성 「면역세포살해」: 사람 *포식·호중구* 반감

### 9) 회충 (연충·선충)
- 태그: `structure:선충` `location:세포외(장관강)` `pathway:소화기` `size:대형` `stage:유충↔성충`
- 준비기 「유충 이행」: 방어·HP↑ / 증상 "장→간→폐 이행"
- 공격기 「흡혈·폐색」 위력10 명중95 `type:물리파괴` 효과[빈혈] 증상 "Löffler 증후군·복통"
- 방어특성 「대형저항」: 사람 *포식소화* 무효(크기)

### 10) 주혈흡충 (연충·흡충)
- 태그: `structure:흡충` `location:세포외(혈관)` `pathway:피부(경피)` `vector:담수달팽이` `size:대형`
- 준비기 「숙주항원 위장」: 방어 대폭↑ / 증상 "숙주단백 코팅"
- 공격기 「충란 육아종」 위력9 명중90 `type:면역매개` 효과[섬유화] 증상 "혈뇨·간 섬유화"
- 방어특성 「항원위장」: 사람 *항체·ADCC* 무효 + 대형

## 7. 사람 진영 자산

### 7-A. 사람 공격기 (면역계 + 약물)
- 선천: 포식소화·산화살균·NET포박·보체옵소닌·보체용해·NK세포독성·인터페론
- 후천: 항체중화·ADCC·세포독성(CTL)·대식세포각성(Th1)·호산구공격(Th2)·호중구소집(Th17)
- 약물: 환경-세포벽/막/단백합성/핵산/엽산/물리 · (확장)항바이러스/항진균/항기생충
- 판정 대상: 패시몬 방어특성 + 패시몬 태그

### 7-B. 사람 방어특성 (면역 제외 · 1차 장벽 + 수동저항)
보스 랜덤 장착. 패시몬 공격을 면역 없이 견디는 장치.

| 사람 방어특성 | 막는 패시몬 공격 | 짝 태그 |
|---|---|---|
| 상피장벽 | 물리파괴(침입) | pathway:피부 |
| 점액섬모 | 물리파괴(호흡기 부착) | pathway:호흡기 |
| 위산 | 물리파괴·독소(경구) | pathway:소화기 |
| 정상균총 | 물리파괴(정착) | reservoir:상재균 |
| 철제한 | 물리파괴(증식) | — |
| 항독소(수동면역) | 독소 | — |

## 8. 판정 흐름 (한 턴)

**패시몬 공격** (표1)
1. 공격기 `type`(관통) → 사람 방어특성과 상성(예: 독소 × 위산 = 반감)
2. 병원체 태그가 사람 방어와 매칭되면 추가 배수
3. `effects` 적용(상태이상 사전 참조) → 데미지/디버프
4. `symptom` 로그에 누적 표시

**사람 공격** (표2)
1. 공격기(면역/약물) → 패시몬 방어특성과 상성(예: 포식 × 피막 = 무효)
2. 패시몬 태그 매칭 추가 배수(예: 항체 × location:세포내 = 무효)
3. effects 적용

**준비기**: 판정 없이 자기 버프. 교체 시 초기화.

## 9. 데이터 구조 요약

```js
// 공격기
MOVES.cholera_toxin = { name:'콜레라독소', power:16, acc:85,
  type:'독소', effects:['탈수','설사'], symptom:'쌀뜨물 설사' }

// 준비기
MOVES.cholera_attach = { name:'소장 상피 부착', kind:'prep',
  effects:[{kind:'buff',stat:'attack',mult:2.0,scope:'battle',target:'self'}],
  symptom:'소장 상피세포 표면에 부착' }

// 상태이상 사전(전투 로직)
STATUS.탈수 = { effect:'hp_dot+def_down', tickPct:8, defDown:20, icon:'💧', desc:'…' }

// 병원체(태그·특성·기술 id 참조)
MONSTERS.cholera = { name:'콜레라', category:'세균',
  tags:{ structure:'그람음성', location:'세포외', pathway:'소화기', size:'미시' },
  ability:'무방비',
  prep:'cholera_attach', learnset:['cholera_toxin', …] }
```

## 10. 미확정 대기
- 준비기 턴 소모 여부(빌드업 vs 즉딜).
- 사람 방어특성 8종을 보스 랜덤 풀로 확정할지.
- 방어특성 2슬롯 허용 여부(결핵=식포융합차단+활성산소중화).
- 상태이상 수치 밸런싱(tickPct·turns 등).
- 증상 누적 UI: 로그 스택 / 사이드 패널 / 전투후 경과 요약 중 무엇.
